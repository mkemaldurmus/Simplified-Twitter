#### **Akka-Http Rest-Api**
create an HTTP Rest API for a simplified twitter app, which will contain functionality for creating, deleting and listing and filtering tweets.

Keys;
- Scala
- Akka-Http
- Unit Testing
- MongoDB 
- Modular Design Pattern
- Docker
